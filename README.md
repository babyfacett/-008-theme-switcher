# 08 色テーマ変更アプリ（ダーク/ライト）

このアプリは、ページ全体の背景色と文字色をライトモードとダークモードで切り替えるデモ用ツールです。Streamlit に CSS を埋め込む方法を学ぶことができます。

## 使い方

1. 以下のコマンドでアプリを起動します。

    ```bash
    streamlit run app.py
    ```

2. 表示されるラジオボタンから「ライト」または「ダーク」を選択すると、ページの背景色と文字色が選択したテーマに応じて変化します。

3. CSS の埋め込みは `unsafe_allow_html=True` を指定して `st.markdown` でスタイルを挿入することで実現しています。テーマ切り替えの仕組みを学習する例としてご利用ください。
